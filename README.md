# Tamga Landing Page

This repository contains the official landing page for **Tamga** - an open-source, local-first, zero-knowledge digital vault.

## Project Goal
To provide a beautiful, modern, and dark-themed web presence for Tamga, highlighting its core features:
- Password Management
- Desktop OTP (TOTP)
- Google Authenticator Migration
- Developer `.env` Security

## Technology Stack
- **React 19** 
- **Vite**
- **React Router v7**
- **Vanilla CSS** (Custom Design System with Glassmorphism)
- **Lucide React** (Iconography)

## Development Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/osmn-byhn/tamga-web.git
   cd tamga-web
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
